gaia_10arcs HATS Margin Catalog

⚠ ⚠Note: This is auxiliary data. It is a margin catalog used for cross-matching and is not intended to be used as a standalone catalog. ⚠ ⚠

This is the 10 arcsecond margin catalog for Gaia DR3.

Access the catalog

We recommend the use of the LSDB Python framework to access HATS catalogs. LSDB can be installed via pip install lsdb or conda install conda-forge::lsdb, see more details in the docs. The following code provides a minimal example of reading this margin catalog:

import lsdb

catalog = lsdb.read_hats("<PATH>")

This catalog is represented as an Apache Parquet dataset and can be accessed with a variety of tools, including pandas, pyarrow, dask, Spark, DuckDB.

File structure

This catalog is represented by the following files and directories:

  • gaia_10arcs/ — margin catalog directory
    • dataset/ — Apache Parquet dataset directory
      • ... parquet metadata and data files in sub directories ...
    • hats.properties — textual metadata file describing this margin catalog
    • partition_info.csv — CSV file with a list of catalog HEALPix tiles
